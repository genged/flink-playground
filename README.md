# Flink Playground

A personal repository with some helpful flink tests.

Currently includes:
 * Usage of EsInputSource from Elasticsearch Hadoop (https://github.com/elastic/elasticsearch-hadoop) with
   Flink Hadoop Compatibility (https://ci.apache.org/projects/flink/flink-docs-stable/dev/batch/hadoop_compatibility.html)
 * Gradle based Flink project (Instead of Maven)
